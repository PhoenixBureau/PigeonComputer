To set up the working environment start in the toplevel directory and...


    virtualenv virt-env
    source ./virt-env/bin/activate
    cd thirdparty
    tar xzf myhdl-0.7.tar.gz
    cd myhdl-0.7
    python setup.py install
    cd ..
    rm -rf myhdl-0.7
    cd ..
    pip install -r requirements.text 

Then run with:

    python pyavrasm.py


To generate the asm.py file from the pigeon_firmware.rst file:

    cd assembler
    ./rest2asm.py ../docs/pigeon_firmware.rst > asm.py
